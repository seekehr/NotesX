# NotesX
A note-taking website made by me. Some stuff it uses:
- React.js
- https://ui.shadcn.com
- axios
- express.js (backend)
- mysql database

Backend server runs on port 5000 (e.g http://localhost:5000) and frontend on 3000.

Enjoy!
